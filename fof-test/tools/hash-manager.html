密码修改处理流程
  1. 管理员生成密码重置邀请链接发送给用户
  2. 用户点击链接进入密码重置页面
  3. 用户输入新密码并生成Shared Secret
  4. 用户将Shared Secret告知管理员
  5. 管理员手动更新password.csv文件中的Shared Secret
  6. 运行json_upload.py自动处理数据迁移
