Shared Secret文件管理工具

密码修改处理流程
  1. 管理员在下方工具中输入用户名，系统自动匹配预设的8位编码
  2. 管理员将生成的邀请链接发送给用户
  3. 用户点击链接进入密码重置页面
  4. 用户输入新密码并生成Shared Secret
  5. 用户将Shared Secret告知管理员
  6. 管理员手动更新用户数据文件中的Shared Secret
